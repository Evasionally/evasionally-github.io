CerealBoxd

Overview

This project is a cereal review social media intended to parody the popular film app LetterBoxd. Within this application user can like, rate, and comment on their favorite cereals as well as suggest new cereals to be added to the platform. After signing up, users can swap between a web app optimized for desktop use, or a mobile version for use on android devices. Currently, the database stores hudnreds of cereals for users to review.
